# CopyAppearanceToClipboard
{: .no_toc }

Copies the current player's appearance data to the clipboard.

**Example:**
```lua
local Bridge = exports['community_bridge']:Bridge()
Bridge.Clothing.CopyAppearanceToClipboard()
```
