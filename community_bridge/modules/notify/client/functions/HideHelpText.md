# HideHelpText
{: .no_toc }

Hides the currently displayed help text.

## Syntax

```lua
Bridge.Notify.HideHelpText()
```

## Example

```lua
local Bridge = exports['community_bridge']:Bridge()
Bridge.Notify.HideHelpText()
```
