# hexToRgb
{: .no_toc }

Converts hex color to RGB values.

## Syntax

```lua
Bridge.Accessibility.hexToRgb(hex)
```

## Parameters

**hex:** `string`  
Hex color code (e.g., "#FF0000")

## Returns

**table**  
RGB color values

## Example

```lua
local Bridge = exports['community_bridge']:Bridge()
local rgb = Bridge.Accessibility.hexToRgb("#FF0000")
```
